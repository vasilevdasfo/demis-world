5 способов добавить картинки в .pptx — через Claude Code.

Если используешь anthropic-skills:pptx, но презентация выходит без иллюстраций — вот рецепты от самого быстрого (URL) до самого продуманного (AI-генерация с брендингом). Каждый — с готовым промптом который можно скопировать в Claude Code.

01
По URL (самый быстрый)
30 сек

Ты даёшь Claude ссылку на картинку — он её скачивает, вставляет в нужный слайд. Работает с любым публичным изображением: Wikipedia, vda.vc, GitHub README, твой собственный сайт.

Промпт:

Сделай pptx на 5 слайдов про "Мой стартап":
- Slide 1 (title): название + фото основателя, возьми https://vda.vc/assets/me.jpg
- Slide 2 (problem): иконка проблемы, возьми https://heroicons.com/icons/alert.svg
- Slide 3 (market): график роста, сгенерируй matplotlib через xlsx skill
- Slide 4 (team): 3 фото → URL ниже
- Slide 5 (ask): только текст, без картинок
Положи в ~/presentations/startup.pptx
Важно: URL должен быть прямой ссылкой на файл (.jpg/.png/.svg/.webp), а не на страницу. Если картинка на Google Drive — сначала получи shareable link «Anyone with link».
02
Из локальной папки
20 сек

У тебя уже есть картинки на маке/ПК — укажи путь, Claude вставит.

Подготовка: собери все изображения в одну папку, назови понятно.

~/presentations/assets/
  ├── logo.png
  ├── founder.jpg
  ├── market-chart.png
  └── team-grid.jpg

Промпт:

Сделай pptx на 6 слайдов. Все картинки — из ~/presentations/assets/:
- Slide 1: logo.png по центру, текст "Startup Name — 2026"
- Slide 2-5: по одному слайду на каждый из остальных файлов +
  заголовок (придумай) + 3 bullet-а под каждой картинкой
- Slide 6: speaker notes со ссылками на источники
Сохрани в ~/presentations/deck-v1.pptx
03
Со стоков (Unsplash / Pexels)
45 сек

Нет своих картинок — Claude найдёт подходящие на Unsplash или Pexels и вставит. Бесплатно, без атрибуции (technically Unsplash требует credit, но для внутренних презентаций норм).

Промпт:

Сделай pptx 8 слайдов про "AI-агенты для фин-консалтинга".
Для каждого слайда — релевантная картинка с Unsplash:
- Slide 1: hero-image про будущее, technology, abstract
- Slide 2: problem — hands struggling with papers / confusion
- Slide 3: solution — clean office, screens, calm
- Slide 4: demo — laptop with data dashboard
- Slide 5: market — city skyline at night
- Slide 6: team — diverse group at whiteboard
- Slide 7: testimonials — smiling professionals
- Slide 8: ask — open door, path forward
Используй Unsplash API (https://source.unsplash.com/1200x675/?keyword)
Сохрани в ~/presentations/ai-agents.pptx

Unsplash

Бесплатные высококачественные фото. Прямой URL по keyword.

source.unsplash.com/?office

Pexels API

Альтернатива. Требует key (бесплатно, 200 запросов/час).

pexels.com/api
04
AI-генерация (DALL-E / Ideogram / Flux)
2-3 мин

Нужны изображения которых нет в природе — AI генерит по описанию. Лучший способ если делаешь брендовый deck с уникальным стилем.

Сервисы:

  • DALL-E 3 (OpenAI API) — $0.04 за картинку 1024×1024, отлично с текстом на изображении
  • Ideogram — сильный с типографикой, логотипами
  • Flux-1.1 Pro — фотореализм, детали
  • Stability AI / Stable Diffusion — опенсорс, дешевле, нужен скилл промптинга

Промпт (через DALL-E):

Сначала сгенерируй 3 hero-изображения для deck'а через OpenAI API
(ключ в ~/.env OPENAI_API_KEY):

1. "minimalist gold on black composition, abstract network lines
   forming a flower pattern, editorial luxury magazine style,
   Playfair Display typography overlay '01 Identity'"

2. "same style, but representing growth — data flowing upward,
   dark gradient, gold accents, no people, no text"

3. "same series, team — 3 silhouettes in circle, warm golden
   light between them, editorial tone, cinematic"

Сохрани в ~/presentations/assets/ai-gen-01.png, 02, 03.

Потом собери pptx 6 слайдов используя их как hero-картинки на
title, transition-slides и финальном CTA.
Pro-tip: чтобы серия была в едином стиле — в каждом промпте повторяй стилистические фиксированные части («same style», «gold on black», «Playfair typography»). Варьируй только сюжет. Иначе картинки получатся разнородными и deck будет смотреться хаотично.
05
Скриншоты — твой сайт, UI, графики
1 мин

Презентуешь продукт — нужны скрины UI. Claude может открыть URL или локальный файл через Preview MCP и сделать скриншот.

Промпт:

Используй Preview MCP:
1. Открой https://vda.vc, сделай скриншот hero + closing CTA (375×812 mobile + 1440 desktop)
2. Открой https://demis.world/onboarding, скриншот полной страницы
3. Открой локальный ~/projects/my-landing/index.html, скриншоты как выше

Картинки сохрани в ~/presentations/assets/screenshots/.

Потом pptx deck 5 слайдов:
- Slide 1: "Что мы делаем" — screenshot vda.vc hero
- Slide 2: "Как это работает" — screenshot onboarding
- Slide 3: "Новый клиентский лендинг" — мой my-landing
- Slide 4: mobile-версии всех 3
- Slide 5: ссылки и QR-коды

Сохрани в ~/presentations/product-demo.pptx

Шпаргалка — какой метод когда

Быстро наотмашь

Внутренняя встреча, 10 минут на подготовку → метод 01 (URL) или 03 (Unsplash). Unsplash даёт более атмосферные картинки чем случайный URL.

Свои данные / UI

Продукт/кейс/метрики → метод 02 (локальные файлы) для существующих + 05 (скриншоты) для UI и дашбордов.

Клиентский deck с брендингом

Презентация клиенту за деньги → метод 04 (AI-генерация) с фиксированным стилем. Инвестировать 30 минут в 5-7 hero-картинок, дальше микс с 02.

Инвесторский deck

Для инвесторов — 02 + 05 + 04: свои данные в скриншотах + команда в фото + стилизованные hero от AI. Никогда не только сток — выглядит generic.

5 частых ошибок новичков

  • Смешивать стили. Фото + 3D-рендер + скетч в одном deck = хаос. Выбери один визуальный язык.
  • Низкое разрешение. Для pptx минимум 1280×720 на full-screen картинку. Меньше — будет пиксельно на большом экране.
  • Текст поверх сложной картинки. Гарантированно нечитаемо. Либо overlay-затемнение (rgba(0,0,0,0.5)), либо простой фон.
  • Забыть про темную/светлую версию. На проекторах часто засвечивает — проверь deck в темной комнате И при ярком свете перед отправкой.
  • Переборщить. Картинка не нужна на каждом слайде. Некоторые слайды лучше оставить чистым текстом на градиенте.

Готовый рецепт — твоя первая презентация с картинками (15 минут)

1. В Claude Code попроси:
   "Сделай /skills/morning-brief.md как referenced в https://demis.world/plans/noname-pl.
   Потом — новая задача: pptx deck 7 слайдов про мой проект 'X'
   (где X = то над чем ты сейчас работаешь).

   Дизайн: dark theme, gold accents, Playfair+Inter.
   Картинки: 3 от DALL-E (hero-style как описано в https://demis.world/skills/images-in-presentations),
   2 скриншота с моего сайта (URL: ...),
   2 Unsplash по темам (office, handshake).

   Speaker notes по 3 строки на каждом.
   Экспорт в pptx + pdf.
   Положи в ~/presentations/."

2. Дождись отработки (2-3 мин на DALL-E + 1 мин на остальное).
3. Открой .pptx в Keynote / PowerPoint — поправь руками что не нравится.
4. Экспорт pdf → клиенту / Telegram.
