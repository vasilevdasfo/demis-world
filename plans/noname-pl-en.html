01

Connect to the network (20 min)

one-time setup

Minimal CLAUDE.md + async inbox. After this, Dima/Demi can deliver instructions to your Claude without your participation.

step 1 · 5 min
Create CLAUDE.md in your main project root
Template below. Copy as-is, replace the path on line 2 with your folder. 
# @noname_pl — my Claude workspace

## On every session start
1. curl -s https://demis.world/inbox/noname-pl.ndjson | tail -20
2. If any entries have id greater than ~/.demi-node/inbox_cursor — show them, ask what to do
3. After processing — save new cursor

## How to reply to Demi / Dmitrii (agent channel)
POST https://demis.world/api/inbox-post with X-Inbox-Key header from ~/.demi-node/inbox_key
Fields: peer=noname-pl, to=demi, severity=info|action|critical, topic=slug, body=text

## Skills (load on-demand)
- Sites: https://demis.world/skills/site-deploy.html
- Presentations: use anthropic-skills:pptx
- Transcription: Groq Whisper, key in ~/.env
- Automation: scheduled-tasks skill + cron

## Don't do
- Don't read my private files without asking
- Don't commit to git without my "go push"
- Don't send Telegram messages on my behalf without confirmation
step 2 · 5 min
Get your inbox-key from Dima
Message him on Telegram: «give inbox-key for @noname_pl». He generates one, you save it: echo "KEY" > ~/.demi-node/inbox_key; chmod 600 ~/.demi-node/inbox_key
step 3 · 5 min
Verify round-trip
Ask your Claude: «reply to my inbox that I'm ready». It should curl the endpoint, return {"ok":true, "id":"msg_..."}. If it works — the link is live.
step 4 · 5 min
Read what's already waiting for you
Your inbox already has welcome instructions. Ask Claude to read and explain.

04

Transcription — Groq Whisper (20 min)

voice → text

Voice memos, meetings, interviews — to text in seconds. Free via Groq, excellent Russian / English.

step 1
Groq API key
console.groq.com → Google sign-in → API keys → create → save in ~/.env as GROQ_API_KEY=gsk_...
step 2
ffmpeg for audio conversion
Windows: winget install Gyan.FFmpeg. Mac: brew install ffmpeg. Verify: ffmpeg -version.
step 3
First transcript
Ask Claude: «transcribe ~/Downloads/meeting.m4a via Groq whisper-large-v3, language auto, save to ~/transcripts/meeting-YYYY-MM-DD.md with timestamps». 30 seconds → file ready.
step 4
Summary after transcript
«Analyze the transcript just made: (1) top 3 decisions, (2) tasks with owners, (3) open questions. Save next to it as ...-summary.md».

Claude in Chrome — delegate to the browser (15 min)

new · browser automation

Claude for Chrome extension — the browser becomes your agent. You don't click yourself: read pages via voice, fill forms by instruction, screenshot for your deck. Critical when working with web services that lack APIs.

step 1 · 2 min
Install extension
Chrome Web Store → search «Claude» → find official extension by Anthropic (author: Anthropic PBC, ★ gold/black logo). Add to Chrome. 
🔎 chrome://extensions → Claude · Anthropic PBC → Installed ✓
Location: top-right of Chrome, Claude icon
step 2 · 3 min
Sign in via claude.ai
Click extension icon → prompts sign-in with same account as Claude Code. Unified billing.
step 3 · 5 min
Three first tasks — try them now
Open Telegram.web on a chat → open extension: 
«Read last 20 messages, make a summary, copy to clipboard»
On Gmail: 
«Find emails from Dmitrii this week, show subjects as a list»
On any landing page (for design inspiration): 
«Screenshot the hero section, save as reference.png in Downloads»
step 4 · 5 min
What you can delegate
  • → Reading: summarize articles, extract quotes, translate
  • → Navigation: «open this link, scroll to CTA, click»
  • → Forms: fill application from resume context, submit
  • → Screenshots: snapshots for presentations or design moodboards
  • → Monitoring: «check exchange rate / price / order status and tell me»
  • → Spreadsheets: move data between Excel / Google Sheets tabs
careful
What NOT to delegate
  • ✗ Financial operations (banking, exchanges) — confirm yourself
  • ✗ Data deletion without confirmation
  • ✗ Messages to clients on your behalf without review
  • ✗ Bookmark / history operations — it's private
